git 的使用：

第一次发布项目(本地-远程)
git add . 将本地文件 添加到暂存区
git commit -m "注释内容 " 将暂存区的内容提交到本地仓库(本地分支）
git push -u origin master


提交(本地-远程)
git add . 将本地文件 添加到暂存区
git commit -m "注释内容 " 将暂存区的内容提交到本地仓库(本地分支）
git push  origin master


第一次下载项目(远程-本地)
git clone git@github.com:xuqiaoZzz/xuqiaoZzz.github.io.git

更新(远程-本地)
git pull
